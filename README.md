# ChineseAddressPicker（中国地址选择器）

[English](./README-en.md)

## 这是啥？

ChineseAddressPicker（中国地址选择器）是一个安卓控件，通过这个控件你可以轻松选择一个中国的地址。

## 特性

* 省市区三级级联

* 滚动控件选择地址

* 平滑的动画效果

* 可以按照读音对地址进行排序，亦可按照官方行政区域ID对地址进行排序

## 技术说明

* 最低API支持: 5

* 支持所有的屏幕尺寸及分辨率

## 请求帮助

地址数据来源于互联网，我不知道这是否是最新的，所以很欢迎你帮我修改地址数据文件，谢谢。
